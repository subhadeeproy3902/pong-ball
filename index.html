PaddleBall
Terminal Arcade.

Physics-based paddleball that lives in your terminal. Delta-time movement, neon particles, 5 difficulty phases, persistent leaderboard — all in a single binary.

Install View Source
paddle-ball play --mode arcade

What's inside

Built to feel like an arcade cabinet.

Every detail is intentional — from the physics to the particle trails to the phase-transition banners.

⚡
Delta-Time Physics
All movement is frame-rate independent. The ball feels identical at 30fps or 120fps, and paddle spin transfers onto the ball for real skill expression.
physics.go
✨
Particle System
Wall impacts scatter glyphs, paddle hits flash white, score pops drift upward. The game-over explosion uses 16 characters flying in all directions.
particles.go
🏆
Persistent Leaderboard
Every game is saved atomically to ~/.paddle-ball/scores.json with mode, streak, duration, and timestamp. Browse it in-game or with paddle-ball scores.
store.go
🎨
4 Color Themes
Neon Arcade, Monochrome, Sunset, and Ocean Night. Switch mid-game with T — your choice is saved to config for next time.
ui/theme.go
🔋
5 Difficulty Phases
Auto-escalating from Warm Up to INSANE. Each phase transition fires an animated banner sweep, border pulse, and speed burst.
scoring.go
📦
Single Static Binary
Zero runtime dependencies. Installs in seconds via Homebrew, apt, Snap, Scoop, WinGet, or a single curl command.
goreleaser

Game Modes

Four ways to play.

Pick your flavour from the title screen or jump straight in with a flag.

🎯
Classic
One life. No power-ups. Pure physics and reflexes against an ever-accelerating ball. The purest test of skill.
🕹️
Arcade
3 lives, power-ups drop every 7 catches, streaks multiply your score. The full experience with all features enabled.
🌊
Zen
Infinite lives, no game over. Just you and the ball, forever. Power-ups still spawn. Perfect for learning the physics.
⏱️
Time Trial
60 seconds. Infinite lives. Score as many points as possible before the clock hits zero. Every streak multiplier counts.

Installation

One command. Any platform.

Pre-built static binaries — no Go required.

macOS / Linux curl -fsSL https://raw.githubusercontent.com/subhadeeproy3902/paddle-ball/main/install.sh | bash

Downloads the latest release binary for your OS and arch, installs to /usr/local/bin.

tap brew tap subhadeeproy3902/paddle-ball
install brew install paddle-ball
upgrade brew upgrade paddle-ball
quick curl -fsSL https://packages.paddle-ball.dev/install.sh | sudo bash
then sudo apt install paddle-ball
install sudo snap install paddle-ball
bucket scoop bucket add paddle-ball https://github.com/subhadeeproy3902/scoop-paddle-ball
install scoop install paddle-ball
install winget install subhadeeproy3902.paddle-ball
go go install github.com/subhadeeproy3902/paddle-ball@latest

Requires Go 1.21+. Binary lands in $GOPATH/bin (make sure it's on your PATH).

run docker run --rm -it ghcr.io/subhadeeproy3902/paddle-ball:latest

No install needed — runs directly in any Docker environment. Use a terminal with at least 80×24 cells.
